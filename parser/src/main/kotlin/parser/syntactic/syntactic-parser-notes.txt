     * DIFFERENT KINDS OF EXPRESSIONS
     *
     *   expression     → assigment | READ_INPUT ;
     *   readInput      → READ_INPUT "(" expression ")" ;
     *   assigment      → IDENTIFIER "=" assignment
     *                | comparison ;
     *   comparison     → term ( ( ">" | ">=" | "<" | "<=" ) term )* ;
     *   term           → factor ( ( "-" | "+" ) factor )* ;
     *   factor         → unary ( ( "/" | "*" ) unary )* ;
     *   unary          → ( "!" | "-" ) unary
     *   | primary ;
     *   primary        → NUMBER | STRING | "true" | "false" | "null"
     *   | "(" expression ")" | IDENTIFIER | READ_ENV;
     *

     * DIFFERENT KINDS OF STATEMENTS:
     *
     * program        →  declaration* EOF ;
     *
     * declaration    → varDecl
     *                | statement ; // later here: functions, and classes
     *
     * statement      → exprStmt
     *                | ifStmt
     *                | printStmt ;
     *                | block ;
     *
     * block          → "{" declaration* "}" ;
     * ifStmt         → "if" "(" expression ")" statement
                      ( "else" statement )? ;
     * exprStmt       → expression ";" ;
     * printStmt      → "print" expression ";" ;
     * varDecl        → "let" IDENTIFIER VARIABLE_TYPE ( "=" expression )? ";" ;
     *
